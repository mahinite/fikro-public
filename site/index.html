Fikro is a desktop tool for saving ideas the moment they appear, organizing them automatically, and retrieving them when you're ready to create — without digging through notes or spreadsheets.

Windows · Offline-first · No account required

The problem

Good ideas disappear
before they ever get used.

You see something that works — a hook, a video, a post — and you think “I should use this.” But by the time you open a notes app or spreadsheet, it’s already lost in clutter.

Later, when you want to create, you don’t have a shortage of ideas — you have too many, poorly organized, and no clear way to pick one without overthinking.

Features

Built for capturing thought
without breaking flow.

✦

Hook-first capture

Every idea starts with a short hook — a single line that captures the core thought before it’s lost. Fast capture, no structure required.

◈

Pattern organization

Organize ideas using color-coded patterns. Not folders or complex hierarchies — just simple categories that match how you think.

⊞

Topic tagging

Add lightweight topic tags to connect related ideas across patterns. Useful for finding themes without rigid structure.

⟳

Random idea picker

When you're stuck, Fikro surfaces a random idea from your active pool. It avoids recently seen ideas so you don’t keep repeating the same ones.

⌘

Global hotkey system

Open Fikro instantly from anywhere using a system-wide hotkey. Capture ideas without switching apps or breaking focus.

◉

System tray mode

Fikro runs quietly in the background and stays ready when needed. Close the window without closing the app.

⇄

Local HTTP API

Fikro exposes a local API so other tools and browser extensions can send ideas directly into it. 

POST http://localhost:13457/add-idea
{ "hook": "What if silence had a texture?" }

How it works

Four moments.
One quiet loop.

01

Summon

Press a hotkey. A small window opens anywhere on your screen.

02

Capture

Write a short hook. Add a pattern. Save instantly.

03

Organise

Ideas are stored locally and automatically organized by pattern and topic.

04

Retrieve

Search or let Fikro surface a random idea when you need inspiration..
